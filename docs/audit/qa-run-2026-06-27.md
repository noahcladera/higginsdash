## Bugs fixed this pass

1. **Book page blank on mobile** — `BookClubPicker` passed `hrefFor` from Server Component; marked `"use client"`.
2. **Add to calendar dead tap** — Link-first open via `?addToCalendar=1`; native fixed sheet; desktop duplicate uses `rendersSheet={false}`.
3. **Booking dialog not opening from slot link** — Server `initialSlotIso` + grid remount key; member native `booking-dialog-sheet`.
4. **More / calendar Radix portals** — Native fixed sheets on mobile WebKit.
