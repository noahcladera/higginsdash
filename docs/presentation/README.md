# Presentation assets — William & Heather

Export diagrams for PowerPoint, Keynote, or Google Slides.

## Quick export (recommended)

1. Open [mermaid.live](https://mermaid.live).
2. Open any file in [`diagrams/`](./diagrams/) (e.g. `02-before-after.mmd`).
3. Copy the full file contents into the editor.
4. **Actions → Export → PNG** or **SVG**.
5. Insert the image on your slide.
